## [+] 启用或禁用设置

您可以直接通过 VS Code 设置自定义 ATM 的行为。按您的喜好精确调整体验：

### 外观与动画
- `atm.cursor.animation` — _切换光标动画。_
- `atm.breadcrumbs.animation` — _切换面包屑路径动画。_
- `atm.files.animation` — _切换文件资源管理器动画。_

### 代码质量
- `atm.lint.enableSuggestions` — _启用基于建议的 Lint 快速修复（例如，将 <code>==</code> 改为 <code>===</code>）。_
- `atm.lint.showOperationalWarnings` — _将 ESLint 运行警告显示为诊断信息。_
- `atm.codeSpell.customWords` — _添加到 ATM Code-Spell 词典的自定义单词。_

### 安全与隐私
- `atm.envLens.renderMode` — _<code>.env</code> 值的隐藏方式：<code>blur</code> 保持轮廓可见，<code>mask</code> 完全隐藏文本。_
- `atm.envLens.revealDurationMs` — _已显示 env 值保持可见的时长。_
- `atm.envLens.ultraSecureMode` — _禁用所有悬停显示操作以实现最高安全性。_

### 格式化
- `atm.formatter.enable` — _为 TypeScript、TSX、CSS、Astro 和 JSON 文件启用 ATM Formatter。_
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _与 Prettier 兼容的格式化选项。_
- `atm.formatter.ignorePath` — _忽略文件的路径（默认 <code>.atmignore</code>）。_

### 开发服务器
- `atm.goLive.port` — _实时服务器的端口；如果忙，自动使用下一个空闲端口。_
- `atm.goLive.host` — _Go Live 绑定的主机接口（使用 <code>0.0.0.0</code> 可在局域网暴露）。_
- `atm.runDev.packageManager` — _用于运行开发脚本的包管理器（<code>auto</code>、<code>bun</code>、<code>npm</code>、<code>pnpm</code>、<code>yarn</code>）。_
- `atm.runDev.autoOpen` — _开发服务器打印本地 URL 后自动打开浏览器。_

### 翻译
- `atm.translate.provider` — _主要翻译提供商（auto、Google Unofficial、MyMemory、LibreTranslate、DeepL、Google Cloud）。_
- `atm.translate.fallbackProviders` — _主提供商失败时使用的回退顺序。_
- `atm.translate.deepLEndpoint` — _DeepL 端点类型（<code>free</code> 或 <code>pro</code>）。_

### Git 提交
- `atm.commitEmoji.outputType` — _插入表情符号或提交代码（例如 <code>:sparkles:</code>）。_
- `atm.commitEmoji.insertPosition` — _插入表情的位置：<code>start</code>、<code>end</code> 或 <code>cursor</code>。_
- `atm.commitEmoji.addCustomEmoji` — _将自定义表情添加到选择器中。_
