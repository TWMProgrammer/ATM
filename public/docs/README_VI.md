  <h3>🏧 Hơn 30 tiện ích và cấu hình đã sẵn sàng.</h3>
  <p>
    <strong>ATM</strong> là bộ công cụ VS Code đa ngôn ngữ tất cả trong một, thay thế hơn 30 công cụ riêng lẻ bằng một gói thống nhất, được thiết kế đẹp mắt. Từ kiểm tra chính tả inline và bảo mật <code>.env</code> thông minh đến ảnh chụp mã đẹp, trình duyệt nhúng, máy chủ dev và công cụ Git nâng cao — mọi thứ bạn cần, không có phần thừa. Nhanh như ma. Hoạt động trên VS Code, Cursor, Windsurf, Trae.ai và Kiro.
  </p>

## [+] Bật hoặc tắt cài đặt

Bạn có thể tùy chỉnh hành vi của ATM trực tiếp trong cài đặt VS Code. Điều chỉnh trải nghiệm đúng theo cách bạn muốn:

### Giao diện và hiệu ứng
- `atm.cursor.animation` — _Bật hoặc tắt hoạt ảnh con trỏ._
- `atm.breadcrumbs.animation` — _Bật hoặc tắt hoạt ảnh đường dẫn breadcrumbs._
- `atm.files.animation` — _Bật hoặc tắt hoạt ảnh trình khám phá tệp._

### Chất lượng mã
- `atm.lint.enableSuggestions` — _Bật Quick Fixes Lint dựa trên gợi ý (ví dụ, từ <code>==</code> sang <code>===</code>)._
- `atm.lint.showOperationalWarnings` — _Hiển thị cảnh báo hoạt động ESLint dưới dạng chẩn đoán._
- `atm.codeSpell.customWords` — _Các từ tùy chỉnh được thêm vào từ điển ATM Code-Spell._

### Bảo mật và quyền riêng tư
- `atm.envLens.renderMode` — _Cách ẩn giá trị <code>.env</code>: <code>blur</code> giữ hình dạng hiển thị, <code>mask</code> ẩn hoàn toàn văn bản._
- `atm.envLens.revealDurationMs` — _Thờ gian giá trị env được hiển thị sau khi mở._
- `atm.envLens.ultraSecureMode` — _Vô hiệu hóa mọi thao tác mở bằng hover để đảm bảo an toàn tối đa._

### Định dạng
- `atm.formatter.enable` — _Bật ATM Formatter cho file TypeScript, TSX, CSS, Astro và JSON._
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _Tùy chọn định dạng tương thích với Prettier._
- `atm.formatter.ignorePath` — _Đường dẫn đến tệp ignore (mặc định <code>.atmignore</code>)._

### Máy chủ phát triển
- `atm.goLive.port` — _Cổng cho máy chủ live; cổng tự do tiếp theo sẽ được tự động sử dụng nếu bận._
- `atm.goLive.host` — _Giao diện host mà Go Live liên kết (dùng <code>0.0.0.0</code> để chia sẻ trong mạng LAN)._
- `atm.runDev.packageManager` — _Trình quản lý package dùng để chạy dev script (<code>auto</code>, <code>bun</code>, <code>npm</code>, <code>pnpm</code>, <code>yarn</code>)._
- `atm.runDev.autoOpen` — _Tự động mở trình duyệt khi máy chủ dev in ra URL cục bộ._

### Dịch thuật
- `atm.translate.provider` — _Nhà cung cấp dịch thuật chính (auto, Google Unofficial, MyMemory, LibreTranslate, DeepL, Google Cloud)._
- `atm.translate.fallbackProviders` — _Thứ tự fallback khi nhà cung cấp chính thất bại._
- `atm.translate.deepLEndpoint` — _Loại endpoint DeepL (<code>free</code> hoặc <code>pro</code>)._

### Commit Git
- `atm.commitEmoji.outputType` — _Chèn ký tự emoji hoặc mã commit (ví dụ <code>:sparkles:</code>)._
- `atm.commitEmoji.insertPosition` — _Vị trí chèn emoji: <code>start</code>, <code>end</code> hoặc <code>cursor</code>._
- `atm.commitEmoji.addCustomEmoji` — _Thêm emoji tùy chỉnh vào bộ chọn._

> 💡 **Mẹo chuyên nghiệp:** Mở <code>settings.json</code> (hoặc tìm <code>atm.</code> trong giao diện Cài đặt) để khám phá mọi tùy chọn. Hãy cập nhật tiện ích thường xuyên để có các tính năng mới nhất, cải thiện hiệu năng và bản vá bảo mật.

## ⌨️ Phím tắt

ATM đăng ký sẵn các phím tắt tập trung vào năng suất. Bạn có thể ghi đè bất kỳ phím tắt nào trong <code>File → Preferences → Keyboard Shortcuts</code>.

| Phím tắt | Lệnh | Ngữ cảnh |
| :------- | :------ | :------ |
| <kbd>Alt</kbd> + <kbd>V</kbd> | Xem trước vùng chọn trước thay đổi | Trình chỉnh sửa có vùng chọn |
| <kbd>Alt</kbd> + <kbd>\</kbd> | Commit Emoji (Tự động gán tốt nhất) | Toàn cục |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>.</kbd> | Bộ chọn Commit Emoji | Toàn cục |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>\</kbd> | So sánh mã | Toàn cục |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>\</kbd> | Mở ATM IRA | Toàn cục |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd> | Mở ATM Browser | Toàn cục |
| <kbd>Shift</kbd> + <kbd>Space</kbd> | Voice TTS: Sao chép và đọc | Toàn cục |
| <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>Space</kbd> | Voice TTS: Chọn giọng | Toàn cục |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | Dịch trang tiện ích | Trang tiện ích đang hoạt động |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | Nhạc: Bật/tạm dừng phát | Các trường hợp khác |
| <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>M</kbd> | Mở xem trước MDX | File MDX |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd> | Dịch ghi chú phát hành | Ghi chú phát hành đang hoạt động |
| <kbd>Alt</kbd> + <kbd>R</kbd> | Nhạc: Đài AM ngẫu nhiên | Toàn cục |
| <kbd>Alt</kbd> + <kbd>L</kbd>, <kbd>Alt</kbd> + <kbd>O</kbd> | Go Live: Khởi động máy chủ | Toàn cục |
| <kbd>Alt</kbd> + <kbd>L</kbd>, <kbd>Alt</kbd> + <kbd>C</kbd> | Go Live: Dừng máy chủ | Toàn cục |
| <kbd>Alt</kbd> + <kbd>G</kbd> | Run Dev: Bắt đầu | Toàn cục |
| <kbd>Ctrl</kbd> + <kbd>G</kbd> | Run Dev: Khởi động lại | Toàn cục |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>G</kbd> | Run Dev: Dừng | Toàn cục |

> 🍎 Trên macOS, <kbd>Ctrl</kbd> trở thành <kbd>⌘</kbd> cho các phím tắt chỉ định keybinding macOS. Run Dev restart/stop cố ý dùng <kbd>Ctrl</kbd> trên cả hai nền tảng.

## Cài đặt

Mở _Quick Open_

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Dán lệnh sau và nhấn `Enter`:

```
ext install bastndev.atm
```

### Cách 2 — Xem tiện ích

1. Mở tiện ích (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Tìm kiếm **"ATM"**
3. Nhấn **Cài đặt**
