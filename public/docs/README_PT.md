  <p>
    <strong>ATM</strong> é um kit de ferramentas multilíngue all-in-one para VS Code que substitui mais de 30 ferramentas separadas por um pacote coeso e lindamente projetado. De correção ortográfica inline e segurança inteligente para <code>.env</code> a capturas de código deslumbrantes, navegador embutido, servidores de desenvolvimento e ferramentas Git avançadas — tudo o que você precisa, nada do que não precisa. Rápido como um fantasma. Funciona no VS Code, Cursor, Windsurf, Trae.ai e Kiro.

## [+] Ativar ou desativar configurações

Você pode personalizar o comportamento do ATM diretamente pelas configurações do VS Code. Ajuste a experiência exatamente ao seu gosto:

### Aparência e animações
- `atm.cursor.animation` — _Ativa ou desativa as animações do cursor._
- `atm.breadcrumbs.animation` — _Ativa ou desativa as animações do caminho de breadcrumbs._
- `atm.files.animation` — _Ativa ou desativa as animações do explorador de arquivos._

### Qualidade de código
- `atm.lint.enableSuggestions` — _Ativa Quick Fixes de lint baseados em sugestões (por exemplo, <code>==</code> para <code>===</code>)._
- `atm.lint.showOperationalWarnings` — _Mostra avisos operacionais do ESLint como diagnósticos._
- `atm.codeSpell.customWords` — _Palavras personalizadas adicionadas ao dicionário ATM Code-Spell._

### Segurança e privacidade
- `atm.envLens.renderMode` — _Como os valores <code>.env</code> são ocultados: <code>blur</code> mantém a forma visível, <code>mask</code> oculta o texto completamente._
- `atm.envLens.revealDurationMs` — _Quanto tempo um valor de env revelado permanece visível._
- `atm.envLens.ultraSecureMode` — _Desativa todas as ações de revelação por hover para máxima segurança._

### Formatação
- `atm.formatter.enable` — _Ativa o ATM Formatter para arquivos TypeScript, TSX, CSS, Astro e JSON._
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _Opções de formatação compatíveis com Prettier._
- `atm.formatter.ignorePath` — _Caminho para o arquivo de ignorar (padrão <code>.atmignore</code>)._

### Servidores de desenvolvimento
- `atm.goLive.port` — _Porta do servidor ao vivo; a próxima porta livre é usada automaticamente se estiver ocupada._
- `atm.goLive.host` — _Interface de host à qual o Go Live se vincula (use <code>0.0.0.0</code> para expor na sua LAN)._
- `atm.runDev.packageManager` — _Gerenciador de pacotes usado para executar o script dev (<code>auto</code>, <code>bun</code>, <code>npm</code>, <code>pnpm</code>, <code>yarn</code>)._
- `atm.runDev.autoOpen` — _Abre o navegador automaticamente assim que o servidor dev imprimir sua URL local._

### Tradução
- `atm.translate.provider` — _Provedor de tradução principal (auto, Google Unofficial, MyMemory, LibreTranslate, DeepL, Google Cloud)._
- `atm.translate.fallbackProviders` — _Ordem de fallback usada quando o provedor principal falha._
- `atm.translate.deepLEndpoint` — _Tipo de endpoint DeepL (<code>free</code> ou <code>pro</code>)._

### Commits Git
- `atm.commitEmoji.outputType` — _Insere caractere de emoji ou código de commit (por exemplo, <code>:sparkles:</code>)._
- `atm.commitEmoji.insertPosition` — _Onde inserir o emoji: <code>start</code>, <code>end</code> ou <code>cursor</code>._
- `atm.commitEmoji.addCustomEmoji` — _Adicione suas próprias entradas de emoji ao seletor._

> 💡 **Dica profissional:** Abra <code>settings.json</code> (ou pesquise <code>atm.</code> na UI de Configurações) para explorar todas as opções. Mantenha a extensão atualizada regularmente para garantir que você tenha os recursos mais recentes, melhorias de desempenho e correções de segurança.
