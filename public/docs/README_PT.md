## [+] Ativar ou desativar configurações

Você pode personalizar o comportamento do ATM diretamente pelas configurações do VS Code. Ajuste a experiência exatamente ao seu gosto:

- `atm.cursor.animation` - _Ativa ou desativa as animações do cursor._
- `atm.breadcrumbs.animation` - _Ativa ou desativa as animações do caminho breadcrumbs._
- `atm.files.animation` - _Ativa ou desativa as animações do explorador de arquivos._

> 💡 **Dica Pro:** Mantenha a extensão atualizada regularmente para garantir os recursos mais recentes, melhorias de desempenho e correções de segurança.

## Instalação

Abra o _Quick Open_

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Cole o comando abaixo e pressione `Enter`:

```
ext install bastndev.atm
```

### Método 2 — Vista de extensões

1. Abra as extensões (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Pesquise **"ATM"**
3. Clique em **Instalar**
