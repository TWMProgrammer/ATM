## [+] تفعيل أو تعطيل الإعدادات

يمكنك تخصيص سلوك ATM مباشرة من إعدادات VS Code. اضبط التجربة كما تريد تمامًا:

- `atm.cursor.animation` - _تفعيل أو تعطيل حركات المؤشر._
- `atm.breadcrumbs.animation` - _تفعيل أو تعطيل حركات مسار breadcrumbs._
- `atm.files.animation` - _تفعيل أو تعطيل حركات مستكشف الملفات._

> 💡 **نصيحة احترافية:** حافظ على تحديث الإضافة بانتظام للحصول على أحدث الميزات، تحسينات الأداء وتصحيحات الأمان.

## التثبيت

افتح _Quick Open_

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

الصق الأمر التالي واضغط `Enter`:

```
ext install bastndev.atm
```

### الطريقة الثانية — عرض الإضافات

1. افتح الإضافات (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. ابحث عن **"ATM"**
3. اضغط على **تثبيت**
