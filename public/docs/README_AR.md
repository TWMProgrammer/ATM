  <h3>🏧 +30 إضافة وإعداد جاهز.</h3>
  <p>
    <strong>ATM</strong> هي مجموعة أدوات متعددة اللغات الكل في واحد لـ VS Code تحل محل أكثر من 30 أداة منفصلة في حزمة متماسكة وجميلة التصميم. بدءًا من التدقيق الإملائي المضمن والأمان الذكي لملفات <code>.env</code> ووصولاً إلى لقطات الشيفرة الرائعة والمتصفح المدمج وخوادم التطوير وأدوات Git المتقدمة — كل ما تحتاجه، ولا شيء مما لا تحتاجه. سريع كالشبح. يعمل على VS Code و Cursor و Windsurf و Trae.ai و Kiro.
  </p>

## [+] تفعيل أو تعطيل الإعدادات

يمكنك تخصيص سلوك ATM مباشرة من إعدادات VS Code. اضبط التجربة كما تريد تمامًا:

### المظهر والرسوم المتحركة
- `atm.cursor.animation` — _تبديل رسوم المؤشر المتحركة._
- `atm.breadcrumbs.animation` — _تبديل رسوم مسار breadcrumb المتحركة._
- `atm.files.animation` — _تبديل رسوم مستكشف الملفات المتحركة._

### جودة الشيفرة
- `atm.lint.enableSuggestions` — _تمكين إصلاحات Lint السريعة القائمة على الاقتراحات (على سبيل المثال، من <code>==</code> إلى <code>===</code>)._
- `atm.lint.showOperationalWarnings` — _عرض تحذيرات ESLint التشغيلية كتشخيصات._
- `atm.codeSpell.customWords` — _كلمات مخصصة تُضاف إلى قاموس ATM Code-Spell._

### الأمان والخصوصية
- `atm.envLens.renderMode` — _كيفية إخفاء قيم <code>.env</code>: <code>blur</code> يبقي الشكل مرئيًا، <code>mask</code> يخفي النص بالكامل._
- `atm.envLens.revealDurationMs` — _مدة بقاء قيمة env مُكشفة مرئية._
- `atm.envLens.ultraSecureMode` — _تعطيل جميع إجراءات الكشف عند التمرير لأقصى درجات الأمان._

### التنسيق
- `atm.formatter.enable` — _تمكين ATM Formatter لملفات TypeScript و TSX و CSS و Astro و JSON._
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _خيارات تنسيق متوافقة مع Prettier._
- `atm.formatter.ignorePath` — _المسار إلى ملف التجاهل (الافتراضي <code>.atmignore</code>)._

### خوادم التطوير
- `atm.goLive.port` — _منفذ خادم البث المباشر؛ يُستخدم المنفذ التالي المتاح تلقائيًا إذا كان مشغولًا._
- `atm.goLive.host` — _واجهة المضيف التي يرتبط بها Go Live (استخدم <code>0.0.0.0</code> للتعرض على شبكتك المحلية)._
- `atm.runDev.packageManager` — _مدير الحزم المستخدم لتشغيل سكربت التطوير (<code>auto</code>، <code>bun</code>، <code>npm</code>، <code>pnpm</code>، <code>yarn</code>)._
- `atm.runDev.autoOpen` — _فتح المتصفح تلقائيًا بمجرد أن يطبع خادم التطوير عنوان URL المحلي الخاص به._

### الترجمة
- `atm.translate.provider` — _مزيد الترجمة الأساسي (auto، Google Unofficial، MyMemory، LibreTranslate، DeepL، Google Cloud)._
- `atm.translate.fallbackProviders` — _ترتيب الاحتياط المستخدم عند فشل المزود الأساسي._
- `atm.translate.deepLEndpoint` — _نوع نقطة نهاية DeepL (<code>free</code> أو <code>pro</code>)._

### Commits الخاصة بـ Git
- `atm.commitEmoji.outputType` — _إدراج رمز emoji أو رمز commit (مثل <code>:sparkles:</code>)._
- `atm.commitEmoji.insertPosition` — _أين تُدرج الـ emoji: <code>start</code> أو <code>end</code> أو <code>cursor</code>._
- `atm.commitEmoji.addCustomEmoji` — _أضف إدخالات emoji الخاصة بك إلى منتقي._

> 💡 **نصيحة احترافية:** افتح <code>settings.json</code> (أو ابحث عن <code>atm.</code> في واجهة الإعدادات) لاستكشاف كل الخيارات. حافظ على تحديث الإضافة بانتظام لضمان الحصول على أحدث الميزات وتحسينات الأداء وتصحيحات الأمان.

## ⌨️ اختصارات لوحة المفاتيح

ATM تسجّل اختصارات تركز على الإنتاجية جاهزة للاستخدام. يمكنك تجاوز أي منها في <code>File → Preferences → Keyboard Shortcuts</code>.

| الاختصار | الأمر | السياق |
| :------- | :------ | :------ |
| <kbd>Alt</kbd> + <kbd>V</kbd> | معاينة التحديد قبل التغييرات | المحرر مع تحديد |
| <kbd>Alt</kbd> + <kbd>\</kbd> | Commit Emoji (التعيين التلقائي للأفضل) | عالمي |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>.</kbd> | منتقي Commit Emoji | عالمي |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>\</kbd> | مقارنة الشيفرة | عالمي |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>\</kbd> | فتح ATM IRA | عالمي |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd> | فتح ATM Browser | عالمي |
| <kbd>Shift</kbd> + <kbd>Space</kbd> | Voice TTS: نسخ وقراءة | عالمي |
| <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>Space</kbd> | Voice TTS: اختيار الصوت | عالمي |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | ترجمة صفحة الإضافة | صفحة الإضافة نشطة |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | الموسيقى: تبديل التشغيل/الإيقاف المؤقت | خلاف ذلك |
| <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>M</kbd> | فتح معاينة MDX | ملف MDX |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd> | ترجمة ملاحظات الإصدار | ملاحظات الإصدار نشطة |
| <kbd>Alt</kbd> + <kbd>R</kbd> | الموسيقى: محطة AM عشوائية | عالمي |
| <kbd>Alt</kbd> + <kbd>L</kbd>, <kbd>Alt</kbd> + <kbd>O</kbd> | Go Live: بدء الخادم | عالمي |
| <kbd>Alt</kbd> + <kbd>L</kbd>, <kbd>Alt</kbd> + <kbd>C</kbd> | Go Live: إيقاف الخادم | عالمي |
| <kbd>Alt</kbd> + <kbd>G</kbd> | Run Dev: بدء | عالمي |
| <kbd>Ctrl</kbd> + <kbd>G</kbd> | Run Dev: إعادة تشغيل | عالمي |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>G</kbd> | Run Dev: إيقاف | عالمي |

> 🍎 في macOS، يصبح <kbd>Ctrl</kbd> هو <kbd>⌘</kbd> للاختصارات التي تحدد ربط مفاتيح macOS. إعادة تشغيل/إيقاف Run Dev يستخدمان <kbd>Ctrl</kbd> عن قصد على كلا النظامين.
