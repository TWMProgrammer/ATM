## [+] 설정 활성화 또는 비활성화

VS Code 설정에서 직접 ATM의 동작을 사용자 지정할 수 있습니다. 원하는 경험에 맞게 세밀하게 조정하세요:

### 외관 및 애니메이션
- `atm.cursor.animation` — _커서 애니메이션을 켜거나 끕니다._
- `atm.breadcrumbs.animation` — _breadcrumb 경로 애니메이션을 켜거나 끕니다._
- `atm.files.animation` — _파일 탐색기 애니메이션을 켜거나 끕니다._

### 코드 품질
- `atm.lint.enableSuggestions` — _제안 기반 Lint 빠른 수정을 활성화합니다(예: <code>==</code>를 <code>===</code>로)._
- `atm.lint.showOperationalWarnings` — _운영 중 ESLint 경고를 진단으로 표시합니다._
- `atm.codeSpell.customWords` — _ATM Code-Spell 사전에 추가할 사용자 지정 단어._

### 보안 및 개인정보 보호
- `atm.envLens.renderMode` — _<code>.env</code> 값을 숨기는 방법: <code>blur</code>는 형태를 유지하고 표시, <code>mask</code>는 텍스트를 완전히 숨깁니다._
- `atm.envLens.revealDurationMs` — _표시된 env 값이 보이는 상태를 유지하는 시간._
- `atm.envLens.ultraSecureMode` — _최대 안전을 위해 모든 호버 표시 작업을 비활성화합니다._

### 포맷팅
- `atm.formatter.enable` — _TypeScript, TSX, CSS, Astro, JSON 파일용 ATM Formatter를 활성화합니다._
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _Prettier 호환 포맷팅 옵션._
- `atm.formatter.ignorePath` — _무시 파일 경로(기본값은 <code>.atmignore</code>)._

### 개발 서버
- `atm.goLive.port` — _라이브 서버 포트; 사용 중이면 자동으로 다음 여유 포트가 사용됩니다._
- `atm.goLive.host` — _Go Live가 바인딩할 호스트 인터페이스(LAN에 공개하려면 <code>0.0.0.0</code> 사용)._
- `atm.runDev.packageManager` — _dev 스크립트 실행에 사용할 패키지 관리자(<code>auto</code>, <code>bun</code>, <code>npm</code>, <code>pnpm</code>, <code>yarn</code>)._
- `atm.runDev.autoOpen` — _dev 서버가 로컬 URL을 출력하면 자동으로 브라우저를 엽니다._

### 번역
- `atm.translate.provider` — _기본 번역 제공자(auto, Google Unofficial, MyMemory, LibreTranslate, DeepL, Google Cloud)._
- `atm.translate.fallbackProviders` — _기본 제공자가 실패할 때 사용되는 폴백 순서._
- `atm.translate.deepLEndpoint` — _DeepL 엔드포인트 유형(<code>free</code> 또는 <code>pro</code>)._

### Git 커밋
- `atm.commitEmoji.outputType` — _이모지 문자 또는 커밋 코드 삽입(예: <code>:sparkles:</code>)._
- `atm.commitEmoji.insertPosition` — _이모지 삽입 위치: <code>start</code>, <code>end</code> 또는 <code>cursor</code>._
- `atm.commitEmoji.addCustomEmoji` — _피커에 나만의 이모지 항목 추가._

> 💡 **프로 팁:** <code>settings.json</code>을 열거나(또는 설정 UI에서 <code>atm.</code> 검색) 모든 옵션을 탐색하세요. 최신 기능, 성능 개선, 보안 패치를 받으려면 확장을 정기적으로 업데이트하세요.

| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>.</kbd> | Commit Emoji Picker | Global |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>\</kbd> | Compare Code | Global |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>\</kbd> | Open ATM IRA | Global |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd> | Open ATM Browser | Global |

| <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>Space</kbd> | Voice TTS: Select Voice | Global |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | Translate Extension Page | Extension page active |
