  <h3>🏧 Более 30 расширений и настроек готовы.</h3>
  <p>
    <strong>ATM</strong> — это универсальный многоязычный набор инструментов для VS Code, который заменяет более 30 отдельных инструментов одним целостным, красиво оформленным пакетом. От встроенной проверки орфографии и интеллектуальной защиты <code>.env</code> до эффектных скриншотов кода, встроенного браузера, dev-серверов и продвинутых инструментов Git — всё, что нужно, и ничего лишнего. Быстрый как призрак. Работает в VS Code, Cursor, Windsurf, Trae.ai и Kiro.
  </p>

## [+] Включить или отключить настройки

Вы можете настроить поведение ATM напрямую через настройки VS Code. Подстройте опыт работы точно под свои предпочтения:

### Внешний вид и анимации
- `atm.cursor.animation` — _Включить или отключить анимации курсора._
- `atm.breadcrumbs.animation` — _Включить или отключить анимации пути хлебных крошек._
- `atm.files.animation` — _Включить или отключить анимации проводника файлов._

### Качество кода
- `atm.lint.enableSuggestions` — _Включить быстрые исправления Lint на основе предложений (например, <code>==</code> в <code>===</code>)._
- `atm.lint.showOperationalWarnings` — _Показывать операционные предупреждения ESLint как диагностику._
- `atm.codeSpell.customWords` — _Пользовательские слова, добавленные в словарь ATM Code-Spell._

### Безопасность и приватность
- `atm.envLens.renderMode` — _Как скрываются значения <code>.env</code>: <code>blur</code> сохраняет видимую форму, <code>mask</code> полностью скрывает текст._
- `atm.envLens.revealDurationMs` — _Как долго открытое значение env остаётся видимым._
- `atm.envLens.ultraSecureMode` — _Отключить все действия раскрытия по наведению для максимальной безопасности._

### Форматирование
- `atm.formatter.enable` — _Включить ATM Formatter для файлов TypeScript, TSX, CSS, Astro и JSON._
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _Параметры форматирования, совместимые с Prettier._
- `atm.formatter.ignorePath` — _Путь к файлу игнорирования (по умолчанию <code>.atmignore</code>)._

### Dev-серверы
- `atm.goLive.port` — _Порт для live-сервера; если занят, автоматически используется следующий свободный._
- `atm.goLive.host` — _Интерфейс хоста, к которому привязывается Go Live (используйте <code>0.0.0.0</code>, чтобы открыть доступ в локальной сети)._
- `atm.runDev.packageManager` — _Менеджер пакетов, используемый для запуска dev-скрипта (<code>auto</code>, <code>bun</code>, <code>npm</code>, <code>pnpm</code>, <code>yarn</code>)._
- `atm.runDev.autoOpen` — _Автоматически открывать браузер, как только dev-сервер выведет свой локальный URL._

### Перевод
- `atm.translate.provider` — _Основной провайдер перевода (auto, Google Unofficial, MyMemory, LibreTranslate, DeepL, Google Cloud)._
- `atm.translate.fallbackProviders` — _Порядок fallback, используемый при сбое основного провайдера._
- `atm.translate.deepLEndpoint` — _Тип endpoint DeepL (<code>free</code> или <code>pro</code>)._

### Git-коммиты
- `atm.commitEmoji.outputType` — _Вставлять символ эмодзи или код коммита (например, <code>:sparkles:</code>)._
- `atm.commitEmoji.insertPosition` — _Куда вставлять эмодзи: <code>start</code>, <code>end</code> или <code>cursor</code>._
- `atm.commitEmoji.addCustomEmoji` — _Добавить собственные эмодзи в выбор._

> 💡 **Профессиональный совет:** откройте <code>settings.json</code> (или найдите <code>atm.</code> в интерфейсе настроек), чтобы изучить все параметры. Регулярно обновляйте расширение, чтобы получать новые функции, улучшения производительности и исправления безопасности.

## ⌨️ Горячие клавиши

ATM предоставляет готовые к использованию сочетания клавиш, направленные на продуктивность. Вы можете переопределить любое из них в <code>File → Preferences → Keyboard Shortcuts</code>.

| Сочетание клавиш | Команда | Контекст |
| :------- | :------ | :------ |

| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>.</kbd> | Выбор эмодзи коммита | Глобально |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>\</kbd> | Сравнить код | Глобально |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>\</kbd> | Открыть ATM IRA | Глобально |
| <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd> | Открыть ATM Browser | Глобально |
| <kbd>Shift</kbd> + <kbd>Space</kbd> | Voice TTS: скопировать и прочитать | Глобально |
| <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>Space</kbd> | Voice TTS: выбрать голос | Глобально |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | Перевести страницу расширения | Страница расширения активна |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | Музыка: включить/приостановить воспроизведение | В остальных случаях |
| <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>M</kbd> | Открыть предпросмотр MDX | Файл MDX |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd> | Перевести заметки к релизу | Заметки к релизу активны |
| <kbd>Alt</kbd> + <kbd>R</kbd> | Музыка: случайная AM-станция | Глобально |
| <kbd>Alt</kbd> + <kbd>L</kbd>, <kbd>Alt</kbd> + <kbd>O</kbd> | Go Live: запустить сервер | Глобально |
| <kbd>Alt</kbd> + <kbd>L</kbd>, <kbd>Alt</kbd> + <kbd>C</kbd> | Go Live: остановить сервер | Глобально |
| <kbd>Alt</kbd> + <kbd>G</kbd> | Run Dev: запустить | Глобально |
| <kbd>Ctrl</kbd> + <kbd>G</kbd> | Run Dev: перезапустить | Глобально |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>G</kbd> | Run Dev: остановить | Глобально |

> 🍎 В macOS <kbd>Ctrl</kbd> становится <kbd>⌘</kbd> для сочетаний, указывающих привязку клавиш macOS. Run Dev restart/stop намеренно используют <kbd>Ctrl</kbd> на обеих платформах.

Запустите _Quick Open_

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`

Вставьте следующую команду и нажмите `Enter`:

```
ext install bastndev.atm
```

3. Нажмите **Установить**
