## [+] Activar o desactivar ajustes

Puedes personalizar el comportamiento de ATM directamente desde la configuración de VS Code. Ajusta la experiencia exactamente a tu gusto:

### Apariencia y animaciones
- `atm.cursor.animation` — _Activa o desactiva las animaciones del cursor._
- `atm.breadcrumbs.animation` — _Activa o desactiva las animaciones de la ruta de breadcrumbs._
- `atm.files.animation` — _Activa o desactiva las animaciones del explorador de archivos._

### Calidad del código
- `atm.lint.enableSuggestions` — _Habilita Quick Fixes de lint basados en sugerencias (por ejemplo, de <code>==</code> a <code>===</code>)._
- `atm.lint.showOperationalWarnings` — _Muestra advertencias operativas de ESLint como diagnósticos._
- `atm.codeSpell.customWords` — _Palabras personalizadas añadidas al diccionario ATM Code-Spell._

### Seguridad y privacidad
- `atm.envLens.renderMode` — _Cómo se ocultan los valores de <code>.env</code>: <code>blur</code> mantiene la forma visible, <code>mask</code> oculta el texto por completo._
- `atm.envLens.revealDurationMs` — _Cuánto tiempo permanece visible un valor de env revelado._
- `atm.envLens.ultraSecureMode` — _Desactiva todas las acciones de revelado al pasar el cursor para máxima seguridad._

### Formateo
- `atm.formatter.enable` — _Habilita ATM Formatter para archivos TypeScript, TSX, CSS, Astro y JSON._
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _Opciones de formateo compatibles con Prettier._
- `atm.formatter.ignorePath` — _Ruta al archivo de ignorados (predeterminado <code>.atmignore</code>)._

### Servidores de desarrollo
- `atm.goLive.port` — _Puerto para el servidor en vivo; se usa automáticamente el siguiente puerto libre si está ocupado._
- `atm.goLive.host` — _Interfaz de host a la que se vincula Go Live (usa <code>0.0.0.0</code> para exponer en tu LAN)._
- `atm.runDev.packageManager` — _Administrador de paquetes usado para ejecutar el script de desarrollo (<code>auto</code>, <code>bun</code>, <code>npm</code>, <code>pnpm</code>, <code>yarn</code>)._
- `atm.runDev.autoOpen` — _Abre automáticamente el navegador una vez que el servidor de desarrollo imprime su URL local._

### Traducción
- `atm.translate.provider` — _Proveedor de traducción principal (auto, Google Unofficial, MyMemory, LibreTranslate, DeepL, Google Cloud)._
- `atm.translate.fallbackProviders` — _Orden de respaldo usado cuando falla el proveedor principal._
- `atm.translate.deepLEndpoint` — _Tipo de endpoint de DeepL (<code>free</code> o <code>pro</code>)._

### Commits de Git
- `atm.commitEmoji.outputType` — _Inserta un carácter emoji o código de commit (p. ej. <code>:sparkles:</code>)._
- `atm.commitEmoji.insertPosition` — _Dónde insertar el emoji: <code>start</code>, <code>end</code> o <code>cursor</code>._
- `atm.commitEmoji.addCustomEmoji` — _Añade tus propias entradas emoji al selector._

> 💡 **Consejo pro:** Abre <code>settings.json</code> (o busca <code>atm.</code> en la interfaz de Configuración) para explorar todas las opciones. Mantén la extensión actualizada regularmente para asegurar las últimas funciones, mejoras de rendimiento y parches de seguridad.

Abre _Quick Open_

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`

```
ext install bastndev.atm
```
