## [+] Activar o desactivar ajustes

Puedes personalizar el comportamiento de ATM directamente desde la configuración de VS Code. Ajusta la experiencia exactamente a tu gusto:

- `atm.cursor.animation` - _Activa o desactiva las animaciones del cursor._
- `atm.breadcrumbs.animation` - _Activa o desactiva las animaciones de la ruta breadcrumbs._
- `atm.files.animation` - _Activa o desactiva las animaciones del explorador de archivos._

> 💡 **Tip Pro:** Mantén la extensión actualizada regularmente para contar con las últimas funciones, mejoras de rendimiento y parches de seguridad.

## Instalación

Abre _Quick Open_

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Pega el siguiente comando y presiona `Enter`:

```
ext install bastndev.atm
```

### Método 2 — Vista de extensiones

1. Abre las extensiones (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Busca **"ATM"**
3. Haz clic en **Instalar**
