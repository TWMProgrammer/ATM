## [+] 設定の有効化または無効化

ATM の動作は VS Code の設定から直接カスタマイズできます。好みに合わせて細かく調整できます:

- `atm.cursor.animation` - _カーソルアニメーションを有効化または無効化します。_
- `atm.breadcrumbs.animation` - _パンくずパスのアニメーションを有効化または無効化します。_
- `atm.files.animation` - _ファイルエクスプローラーのアニメーションを有効化または無効化します。_

> 💡 **プロ向けヒント:** 最新機能、パフォーマンス改善、セキュリティ修正を受け取るため、拡張機能を定期的に更新してください。

## インストール

_Quick Open_ を開きます

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

次のコマンドを貼り付けて `Enter` を押します:

```
ext install bastndev.atm
```

### 方法 2 — 拡張機能ビュー

1. 拡張機能を開く (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. **"ATM"** を検索
3. **インストール** をクリック
