  <h3>🏧 +30の拡張機能と設定が準備できています。</h3>
  <p>
    <strong>ATM</strong>は、30以上の個別ツールを1つの統一され美しくデザインされたパッケージに置き換える、多言語対応のオールインワン VS Code ツールキットです。インラインのスペルチェックやインテリジェントな<code>.env</code>セキュリティから、美しいコードスクリーンショット、組み込みブラウザ、開発サーバー、高度な Git ツールまで——必要なものはすべて、不要なものはありません。ghost のように高速です。VS Code、Cursor、Windsurf、Trae.ai、Kiro で動作します。
  </p>

## [+] 設定の有効化または無効化

VS Code の設定から直接 ATM の動作をカスタマイズできます。好みに合わせて体験を微調整しましょう:

### 外観とアニメーション
- `atm.cursor.animation` — _カーソルアニメーションの切り替え。_
- `atm.breadcrumbs.animation` — _パンくずリストパスのアニメーション切り替え。_
- `atm.files.animation` — _ファイルエクスプローラーのアニメーション切り替え。_

### コード品質
- `atm.lint.enableSuggestions` — _候補ベースの Lint クイックフィックスを有効化（例:<code>==</code>を<code>===</code>に）。_
- `atm.lint.showOperationalWarnings` — _運用時の ESLint 警告を診断として表示。_
- `atm.codeSpell.customWords` — _ATM Code-Spell 辞書に追加するカスタム単語。_

### セキュリティとプライバシー
- `atm.envLens.renderMode` — _<code>.env</code>値の隠し方:<code>blur</code>は形を残して表示、<code>mask</code>は完全に隠す。_
- `atm.envLens.revealDurationMs` — _表示した env 値が見える状態を保つ時間。_
- `atm.envLens.ultraSecureMode` — _最大の安全性のため、すべてのホバーによる表示操作を無効化。_

### フォーマット
- `atm.formatter.enable` — _TypeScript、TSX、CSS、Astro、JSON ファイル向けの ATM Formatter を有効化。_
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _Prettier 互換のフォーマットオプション。_
- `atm.formatter.ignorePath` — _無視ファイルのパス（デフォルトは<code>.atmignore</code>）。_

### 開発サーバー
- `atm.goLive.port` — _ライブサーバーのポート。使用中の場合は自動的に次の空きポートが使われる。_
- `atm.goLive.host` — _Go Live がバインドするホストインターフェース（LAN に公開する場合は<code>0.0.0.0</code>を使用）。_
- `atm.runDev.packageManager` — _dev スクリプトの実行に使用するパッケージマネージャー（<code>auto</code>、<code>bun</code>、<code>npm</code>、<code>pnpm</code>、<code>yarn</code>）。_
- `atm.runDev.autoOpen` — _dev サーバーがローカル URL を出力したら自動的にブラウザーを開く。_

### 翻訳
- `atm.translate.provider` — _主な翻訳プロバイダー（auto、Google Unofficial、MyMemory、LibreTranslate、DeepL、Google Cloud）。_
- `atm.translate.fallbackProviders` — _主なプロバイダーが失敗した場合に使用するフォールバック順序。_
- `atm.translate.deepLEndpoint` — _DeepL エンドポイントの種類（<code>free</code>または<code>pro</code>）。_

### Git コミット
- `atm.commitEmoji.outputType` — _絵文字文字またはコミットコードを挿入（例:<code>:sparkles:</code>）。_
- `atm.commitEmoji.insertPosition` — _絵文字の挿入位置:<code>start</code>、<code>end</code>、または<code>cursor</code>。_
- `atm.commitEmoji.addCustomEmoji` — _ピッカーに独自の絵文字エントリーを追加。_

> 💡 **プロのヒント:** <code>settings.json</code>を開くか（または設定 UI で<code>atm.</code>を検索）してすべてのオプションを確認してください。最新機能、パフォーマンス改善、セキュリティパッチを確実に得るために、拡張機能を定期的に更新してください。
