## [+] Settings सक्षम या अक्षम करें

आप VS Code settings से ATM के व्यवहार को सीधे customize कर सकते हैं। Experience को अपनी पसंद के अनुसार ठीक करें:

### उपस्थिति और एनिमेशन
- `atm.cursor.animation` — _कर्सर एनिमेशन टॉगल करें।_
- `atm.breadcrumbs.animation` — _ब्रेडक्रंब पथ एनिमेशन टॉगल करें।_
- `atm.files.animation` — _फ़ाइल एक्सप्लोरर एनिमेशन टॉगल करें।_

### कोड गुणवत्ता
- `atm.lint.enableSuggestions` — _सुझाव-आधारित Lint Quick Fixes सक्षम करें (उदाहरण के लिए, <code>==</code> से <code>===</code>)।_
- `atm.lint.showOperationalWarnings` — _परिचालन ESLint चेतावनियों को डायग्नोस्टिक्स के रूप में दिखाएँ।_
- `atm.codeSpell.customWords` — _ATM Code-Spell शब्दकोश में जोड़े गए कस्टम शब्द।_

### सुरक्षा और गोपनीयता
- `atm.envLens.renderMode` — _<code>.env</code> मान कैसे छुपाए जाते हैं: <code>blur</code> आकार दृश्य रखता है, <code>mask</code> पाठ को पूरी तरह छुपाता है।_
- `atm.envLens.revealDurationMs` — _एक प्रकट env मान कितनी देर तक दिखाई देता है।_
- `atm.envLens.ultraSecureMode` — _अधिकतम सुरक्षा के लिए सभी होवर रिवील क्रियाएँ अक्षम करें।_

### फ़ॉर्मेटिंग
- `atm.formatter.enable` — _TypeScript, TSX, CSS, Astro और JSON फ़ाइलों के लिए ATM Formatter सक्षम करें।_
- `atm.formatter.printWidth`, `atm.formatter.tabWidth`, `atm.formatter.useTabs`, `atm.formatter.semi`, `atm.formatter.singleQuote`, `atm.formatter.trailingComma` — _Prettier-संगत फ़ॉर्मेटिंग विकल्प।_
- `atm.formatter.ignorePath` — _इग्नोर फ़ाइल का पथ (डिफ़ॉल्ट <code>.atmignore</code>)।_

### डेव सर्वर
- `atm.goLive.port` — _लाइव सर्वर का पोर्ट; यदि व्यस्त हो तो अगला मुक्त पोर्ट स्वचालित रूप से उपयोग किया जाता है।_
- `atm.goLive.host` — _होस्ट इंटरफ़ेस जिससे Go Live बाइंड होता है (अपने LAN पर उजागर करने के लिए <code>0.0.0.0</code> का उपयोग करें)।_
- `atm.runDev.packageManager` — _डेव स्क्रिप्ट चलाने के लिए उपयोग किया जाने वाला पैकेज मैनेजर (<code>auto</code>, <code>bun</code>, <code>npm</code>, <code>pnpm</code>, <code>yarn</code>)।_
- `atm.runDev.autoOpen` — _जब डेव सर्वर अपना स्थानीय URL प्रिंट करे तो स्वचालित रूप से ब्राउज़र खोलें।_

### अनुवाद
- `atm.translate.provider` — _प्राथमिक अनुवाद प्रदाता (auto, Google Unofficial, MyMemory, LibreTranslate, DeepL, Google Cloud)।_
- `atm.translate.fallbackProviders` — _जब प्राथमिक प्रदाता विफल हो तो उपयोग किया जाने वाला फ़ॉलबैक क्रम।_
- `atm.translate.deepLEndpoint` — _DeepL एंडपॉइंट प्रकार (<code>free</code> या <code>pro</code>)।_

### Git कमिट
- `atm.commitEmoji.outputType` — _इमोजी वर्ण या कमिट कोड डालें (उदाहरण के लिए <code>:sparkles:</code>)।_
- `atm.commitEmoji.insertPosition` — _इमोजी कहाँ डालें: <code>start</code>, <code>end</code>, या <code>cursor</code>।_
- `atm.commitEmoji.addCustomEmoji` — _पिकर में अपनी खुद की इमोजी प्रविष्टियाँ जोड़ें।_

> 💡 **प्रो टिप:** हर विकल्प को एक्सप्लोर करने के लिए <code>settings.json</code> खोलें (या सेटिंग्स UI में <code>atm.</code> खोजें)। नवीनतम सुविधाओं, प्रदर्शन सुधार और सुरक्षा पैच सुनिश्चित करने के लिए एक्सटेंशन को नियमित रूप से अपडेट रखें।

## ⌨️ कीबोर्ड शॉर्टकट

| <kbd>Alt</kbd> + <kbd>\</kbd> | Commit Emoji (सर्वश्रेष्ठ ऑटो-असाइन) | वैश्विक |

| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | संगीत: चलाएँ/रोकें टॉगल करें | अन्यथा |

| <kbd>Alt</kbd> + <kbd>L</kbd>, <kbd>Alt</kbd> + <kbd>C</kbd> | Go Live: सर्वर रोकें | वैश्विक |
| <kbd>Alt</kbd> + <kbd>G</kbd> | Run Dev: प्रारंभ | वैश्विक |
| <kbd>Ctrl</kbd> + <kbd>G</kbd> | Run Dev: पुनः आरंभ | वैश्विक |
