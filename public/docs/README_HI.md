## [+] Settings सक्षम या अक्षम करें

आप VS Code settings से ATM के व्यवहार को सीधे customize कर सकते हैं। Experience को अपनी पसंद के अनुसार ठीक करें:

- `atm.cursor.animation` - _Cursor animations को चालू या बंद करें._
- `atm.breadcrumbs.animation` - _Breadcrumb path animations को चालू या बंद करें._
- `atm.files.animation` - _File explorer animations को चालू या बंद करें._

> 💡 **Pro Tip:** नए features, performance improvements और security patches पाने के लिए extension को नियमित रूप से update रखें।
