## [+] Einstellungen aktivieren oder deaktivieren

Du kannst das Verhalten von ATM direkt in den VS Code-Einstellungen anpassen. Stelle die Erfahrung genau nach deinen Wünschen ein:

- `atm.cursor.animation` - _Cursor-Animationen aktivieren oder deaktivieren._
- `atm.breadcrumbs.animation` - _Breadcrumb-Pfadanimationen aktivieren oder deaktivieren._
- `atm.files.animation` - _Animationen im Datei-Explorer aktivieren oder deaktivieren._

> 💡 **Profi-Tipp:** Halte die Erweiterung regelmäßig aktuell, um die neuesten Funktionen, Leistungsverbesserungen und Sicherheitsupdates zu erhalten.

## Installation

_Quick Open_ starten

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Füge den folgenden Befehl ein und drücke `Enter`:

```
ext install bastndev.atm
```

### Methode 2 — Erweiterungsansicht

1. Erweiterungen öffnen (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Nach **"ATM"** suchen
3. Auf **Installieren** klicken
